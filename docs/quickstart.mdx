# Quick Start

Get Digi running locally in minutes.

## Prerequisites

- [Bun](https://bun.sh) â‰¥ 1.0
- [Docker](https://docker.com) (for local infrastructure)

## 1. Clone and Install

```bash
git clone https://github.com/digi-run/digi
cd digi
bun install
```

## 2. Start Infrastructure

```bash
docker compose up -d
```

This starts PostgreSQL, Redis, and Mailpit (email testing).

## 3. Configure Environment

```bash
cp .env.example .env
# Edit .env with your configuration
```

## 4. Set Up Database

```bash
cd packages/db && bun run db:push
cd ../..
```

## 5. Create Admin User

```bash
bun run seed:admin
```

Note the email and generated password printed to your terminal.

## 6. Start Development

```bash
bun dev
```

| App | URL |
|-----|-----|
| Landing | http://localhost:3000 |
| Dashboard | http://localhost:3001 |
| Admin | http://localhost:3002 |
| API | http://localhost:4000 |

## 7. Install the CLI

```bash
cd packages/cli && bun run build
# Binary available at packages/cli/dist/digi
```

## 8. Deploy Your First Service

```bash
digi login
digi services create --name my-api --repo https://github.com/you/my-api
digi deploy my-api
```
